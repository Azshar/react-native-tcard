# react-native-tcard

Модуль React-Native для работы с приложение "Pay to Phone" от Т-Банка

## Installation

```sh
npm install react-native-tcard
```

## Usage


```js
import { multiply } from 'react-native-tcard';

// ...

const result = await multiply(3, 7);
```
